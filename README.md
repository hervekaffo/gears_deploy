# Gears (React Native)

This repository contains the Gears React Native (Expo) application.

Quick start

1. Install dependencies

```bash
npm install
# or
# yarn install
```

2. Start Expo

```bash
npx expo start
```

3. Open the app on a device or simulator using the Expo devtools.
